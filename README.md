# Phase-Field Simulation of Compound-Droplet Coalescence with Viscosity Contrast

This repository contains the Python implementation used for the project report  
**"Phase-Field Simulation of Compound-Droplet Coalescence with Viscosity Contrast"**.

The simulations solve the two-dimensional Cahn--Hilliard--Navier--Stokes equations using a Fourier pseudospectral discretization and an ETDRK2 time-stepping scheme. Both constant-viscosity and phase-dependent-viscosity cases are included.

## Repository contents

- `Droplet_Coalescence_CHNS_Simulation.py`  
  Main CHNS simulation code for simple and compound droplet coalescence.

- `Plot_Coalescence_Simulation_Results.py`  
  Plotting and post-processing script used to generate figures from saved simulation data.

## Data availability

The raw simulation data are stored as `.npz` files and can be several hundred MB per simulation. Due to GitHub file-size limitations, these data files are not included in this repository.

The simulation data can be regenerated by running the main simulation script with the parameter choices described in the report. The plotting script expects the corresponding `.npz` output files to be available locally.
